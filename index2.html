About

My project is called Shaquala Quotes App. Random Quotes: Who wrote that Quote?

The user will be able to open the app and see there are two buttons which is called "Get Quotes" and "Clear Quotes" The page will also show there is an "About" title at the top of the page which the user can click on the word and another page will show more details about how the app works.

The user will then click on the "Get Quote" button. After clicking the button, a random of quotes will generate.

The user will read the quotes and take a guess to what author wrote or said that quote.After guessing, the user will then click on the "Reveal Answer" button.The "Reveal Answer" button will generate the correct answer to who wrote or said the quote?

This process will continue until the user is finished with the quotes that was generated. When the user is ready for another set of quotes, First, the user will click on the "Clear Quotes" button to clear the quotes and then repeat to click on "Get Quotes" button to generate a new set of quotes
